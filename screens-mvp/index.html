Screens MVP

AutoZ marketplace visual kit

Telas conceituais do marketplace de veiculos descrito em about.md, desenhadas como mockups de alta fidelidade com linguagem de componentes proxima de React, Tailwind CSS e shadcn/ui.

React Router TanStack Query Zustand Tailwind CSS shadcn/ui React Hook Form Zod
Home

Entrada editorial com busca forte

Hero amplo, busca principal, filtros rapidos, destaques e listagens recentes. A direcao visual foi pensada para landing page de alto impacto sem se afastar do que seria plausivel em Vite + Tailwind.

Search

Resultados com filtros persistentes

Sidebar de filtros, chips ativos, ordenacao e paginação. A tela foi montada para refletir um fluxo natural com TanStack Query e estado global enxuto via Zustand.

Details

Pagina de anuncio com foco em conversao

Galeria grande, preco destacado, seller card, especificacoes, mapa e CTAs de contato. A hierarquia visual segue o tipo de pagina que costuma converter melhor em marketplaces automotivos.

Dashboard

Painel do usuario com acoes claras

Visao resumida de anuncios, favoritos, perfil e atalhos de gestao. A composicao favorece cards compactos, estados e quick actions bem alinhados ao ecossistema shadcn/ui.

Create Listing

Formulario multi-etapas para publicar

Stepper, selects, switches, feedback de validacao e preview lateral. A tela foi guiada por um fluxo plausivel de React Hook Form com Zod e componentes de formulario do stack definido.

Account

Favoritos e perfil na mesma linguagem

Tabs segmentadas, grid de salvos e painel de perfil editavel. A ideia aqui foi consolidar retencao e personalizacao em uma area de conta simples, premium e responsiva.

Os mockups foram gerados como imagens raster e salvos em screens-mvp/assets/. Esta pagina existe apenas para revisar a direcao visual do MVP sem depender do frontend final.
